<h1>A Dictionary App that can save past searches' data with a web API.</h1>
<h3>How it works: </h3>
<ul>
  <li>Storing data in a JSON file (a.json in this case).</li>
  <li>Using a web API to fetch the words' definition and examples.</li>
</ul>
